电脑

打包解包

WinRAR

软件介绍：WinRAR 是一款功能强大的压缩包管理器，它是档案工具RAR在 Windows环境下的图形界面。该软件可用于备份数据，缩减E-mail附件的大小，解压缩从 Internet 上下载的RAR、ZIP及其它类型文件，并且可以新建 RAR 及 ZIP 格式等的压缩类文件。

官网：http://www.winrar.com.cn/

7zip

软件介绍：7-Zip 是一款拥有极高压缩比的开源压缩软件。

官网：https://sparanoid.com/lab/7z/

播放器

VLC

软件介绍：VLC 是一款免费、自由、开源的跨平台多媒体播放器及框架，可播放大多数多媒体文件，以及各类流媒体协议。

官网：https://www.videolan.org/vlc/

Avee Music Player

软件介绍：Avee Music Player 一款支持音频可视化，且可定制可视化样式的音乐播放器。

官网：https://www.aveeplayer.com/

foobar2000

软件介绍：Foobar2000 是一款免费的多功能音频播放器，作者是原Winamp开发公司Nullsoft成员 Peter Pawlowski。除了播放之外，它还支持生成媒体库、转换媒体文件编码、提取CD等功能，是一款功能强大的音频处理工具。

官网：https://www.foobar2000.org/

汉化下载：https://www.cnblogs.com/asionwu

远程操控

ToDesk

软件介绍：ToDesk 是一款多平台远程控制软件，支持主流操作系统Windows、Linux、Mac、Android、iOS跨平台协同操作。

官网：https://www.todesk.com/

向日葵远程控制

软件介绍：向日葵远程控制是上海贝锐信息科技股份有限公司开发的一款提供远程控制服务的软件。

官网：https://sunlogin.oray.com/

Splashtop

软件介绍：Splashtop 是一款由Splashtop公司开发的系列安全远程控制软件。支持4K、60帧/秒的高性能、各种高级安全功能和合规性。可跨操作系统（包括 Windows 和 Mac）实现远程访问和支持。

官网：https://www.splashtop.cn/

虚拟机

VMware Workstation Pro

软件介绍：VMware Workstation Pro 是行业标准桌面 Hypervisor，使用它可在 Windows 或 Linux 桌面上运行 Windows、Linux 和 BSD 虚拟机。

官网：https://www.vmware.com/cn/products/workstation-pro.html

VirtualBox

软件介绍：VirtualBox号称是最强的免费虚拟机软件，它不仅具有丰富的特色，而且性能也很优异。它简单易用，可虚拟的系统包括Windows（从Windows 3.1到Windows 10、Windows Server 2012，所有的Windows系统都支持）、Mac OS X、Linux、OpenBSD、Solaris、IBM OS2甚至Android等操作系统。

官网：https://www.virtualbox.org/
